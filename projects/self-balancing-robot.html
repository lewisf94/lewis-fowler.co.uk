Self-Balancing Robot

← Back to Home

This project demonstrates a two-wheeled self-balancing robot using an Arduino Nano, MPU-6050 gyro, TB6612 motor driver, and classic yellow motors. It was developed to experiment with PID control and gyroscopic feedback.

Key Features

  • MPU-6050 gyro + accelerometer for angle detection
  • PID control loop for stabilization
  • 3D printed frame with ball caster
  • Li-ion battery power with PowerBoost 1000
